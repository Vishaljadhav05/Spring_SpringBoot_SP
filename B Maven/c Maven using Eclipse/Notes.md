# 📘 Maven Basics Notes

## 🧩 What is Archetype?

👉 An **archetype** is a **project template or model** used to quickly create a Maven project.

* It provides a predefined structure for your project 📂
* Saves time ⏳ by generating standard files and folders

### 🔹 Examples:

* 🟢 `maven-archetype-quickstart` → For simple Java projects
* 🌐 `maven-archetype-webapp` → For web applications

---

## 🏢 What is groupId?

👉 The **groupId** is a **unique identifier for an organization or group**.

* Helps differentiate projects from different organizations 🔍
* Usually follows **reverse domain name format** 🌍

### 🔹 Example:

* `www.xyz.com` → `com.xyz`

✔️ Ensures global uniqueness of your project

---

## 📦 What is artifactId?

👉 The **artifactId** is the **unique name of your project**.

* Represents the actual project/module name 🏷️
* Used along with groupId to uniquely identify a project

### 🔹 Naming Examples:

* `MavenFirstProject`
* `mavenFirstProject`
* `maven-first-project`

✔️ Usually written in lowercase with hyphens for readability 👍

---

## 🧠 Quick Summary

* 🧩 **Archetype** → Project template
* 🏢 **groupId** → Organization identifier
* 📦 **artifactId** → Project name

---

✨ These three are the foundation of any Maven project!
